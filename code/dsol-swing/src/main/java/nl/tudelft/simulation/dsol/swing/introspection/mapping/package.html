Defines and contains tablecell renderers and editors to be used by {@see nl.tudelft.tbm.sk.util.introspection.gui.ObjectJTable}. The {@see nl.tudelft.tbm.sk.introspection.gui.mapping.ExpandButtonEditor} controls the 'click-through' functionality of the {@see nl.tudelft.tbm.sk.util.introspection.gui.ObjectJTable}.
